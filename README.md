# Floripa Bar - Site Oficial

Site do restaurante Floripa Bar, especializado em frutos do mar, localizado em Tubarão, SC.

## Tecnologias
- Next.js 16
- React 19
- Tailwind CSS v4
- Shadcn/UI

## Como rodar localmente
1. Instale as dependências: `pnpm install`
2. Rode o servidor de desenvolvimento: `pnpm dev`
3. Acesse `http://localhost:3000`
